## ผมมีข้อมูลสำหรับ Employee_data ที่เป็นรูปแบบของ Excel ของผมอยู่ผมต้องการนำเอาข้อมูลส่วนนี้ขึ้นระบบโดยระบบจะแบ่งการเรียง คอลลัม ดังนี้
- รหัสพนักงาน : สำหรับเก็บข้อมูลรหัสพนักงานที่เชื่อมกับข้อมูล login 
- ตำแหน่ง : สำหรับระบุตำแหน่งการทำงานของพนักงานแต่ละคนในบริษัท
- รหัสาบัตรประชาชน : สำหรับเก็บข้อมูลของรหัสบัตรประชาชนของพนักงาน 13 หลัก
- เพศ : สำหรับเก็บข้อมูลของเพศพนักงาน
- ชื่อจริง : สำหรับเก็บข้อมูลชื่อจริงของพนักงาน
- นามสกุล : สำหรับเก็บข้อมูลนามสกุลของพนักงาน
- ชื่อ - นามสกุล : สำหรับเอาข้อมูล ชื่อ - นามสกุล มารวมกัน
- ชื่อภาษาอังกฤษ : สำหรับเก็บข้อมูลชื่อภาษาอังกฤษของพนักงาน
- ชื่อเล่น : สำหรับเก็บข้อมูลชื่อเล่นของพนักงาน 
- วันเกิด : สำหรับเก็บข้อมูลวันเกิดของพนังกาน 
- เบอร์โทร : สำหรับเก็บข้อมูลเบอร์โทรของพนักงานแต่ละคน
- Email : สำหรับเก็บข้อมูลอีเมลส่วนตัวของพนักงาน
- Email Build : สำหรับเก็บข้อมูลอีเมลที่บริษัทส่งให้พนักงานคนนั้นใช้งานสำหรับทำงาน
- PassWord E-mail Buildme : สำหรับเก็บข้อมูลรหัสผ่านของอีเมลที่ให้พนักงานใช้ในบริษัท
- วันเริ่มงาน : สำหรับเก็บข้อมูลว่าพนักงานคนนี้เริ่มงานวันที่เท่าไหร่
- วันผ่านงาน : สำหรับเก็บข้อมูลว่าพนักงานแต่ละคนผ่านทดลองงานวันที่เท่าไหร่
- วันสิ้นสุด : สำหรับเก็บข้อมูลวันที่พนักงานลาออกจากบริษัท
- สถานะงาน : สำหรับเก็บข้อมูลว่าปัจจุบันพนักงานอยู่ในการทำงานของสถานะไหนจะประกอบไปด้วย > ทำงานอยู่ , ลาออก
- ที่อยู่ : สำหรับเก็บข้อมูลที่อยู่ของพนักงานที่รวมมาเป็นข้อมูลเดียว
- หมู่บ้าน : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล หมู่บ้าน
- อาคาร : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล อาคาร
- ห้องเลขที่ : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล ห้องเลขที่
- ชั้นที่ : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล ชั้นที่
- เลขที่ : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล เลขที่
- ซอย/ตรอก : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล ซอย/ตรอก
- หมู่ที่ : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล หมู่ที่
- ถนน : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล ถนน
- แขวง/ตำบล : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล แขวง/ตำบล
- อำเภอ/เขต : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล อำเภอ/เขต
- จังหวัด : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล จังหวัด
- รหัสไปรษณี : สำหรับเก็บข้อมูลย่อยของพนังกานที่เป็นข้อมูล รหัสไปรษณี
- รูปภาพ : สำหรับเก็บข้อมูลรูปภาพของพนักงานในบริษัท 

**สำหรับข้อมูลนี้ผมอยาก Cursor AI ช่วยออกแบบการเก็บข้อมูลพนักงานตรงนีั้หน่อยครับ ว่าผมจะต้องเก็บข้อมูลยังไง ช่วยออกแบบฐานข้อมูลและเก็บข้อมูลตัวแปลฐานข้อมูลออกมาให้ผมหน่อยนะครับ

## สำหรับในส่วนของข้อมูลพนักงาน : ให้ Cursor AI ช่วยออกแบบระบบของข้อมูลพนักงานโดยกำหนดการทำงานของระบบเเบื้องต้นคือ Role : HR , Admin จะมองเห็นทุกระบบการทำงานของหน้านี้
ถ้าเห็นเป็น Role ปกติให้พนักงานมองเห็นเฉพาะข้อมูลส่วนตัวของตัวเองเท่านั้นไม่อนุญา่ตให้มองเห็นข้อมูลของพนักงานคนอื่น

## ระบบที่จะต้องมีในหน้าข้อมูลพนักงาน :
- Role : HR , Admin จะต้องมองเห็นข้อมูลของพนักงานทุกคนสามารถเปิดดูของใครก็ได้ > ให้ Cursor AI ออกแบบ API สำหรับการเรียกข้อมูลไม่ให้เรียกมาเยอะ ให้เรียกข้อมูลมาเท่าที่ใช้แล้วถ้าต้องการเห็นมากกว่านั้นให้เรียกส่วนอื่นเพิ่มทีหลังเพื่อลดภาระการทำงานของระบบหลังบ้าน
- พนักงานแต่ละคนจะต้องเห็นข้อมูลของตัวเองทั้งหมด 
  1. รหัสบัตรประชาชน
  2. ชื่อ - นามสกุล 
  3. เพศ
  4. วันเกิด
  5. เบอร์โทร
  6. ที่อยู่
  7. อีเมลส่วนตัว
  8. อีเมลของบริษัท
  9. วันเริ่มทำงาน
  10. วันผ่านงาน
  11. ทำงานมาแล้วกี่วัน > ให้ Cursor AI ทำฟีเจอร์สำหรับการคำนวณข้อมูลวันตั้งแต่วันที่เริ่มทำงานถึงปัจจุบันพนักงานคนนี้ทำงานมาแล้วกี่วัน
  12. ข้อมูลสถิติวันลา
  13. ข้อมูลสถิติวัน WFH
- ในส่วนนี้ให้แสดงรูปภาพของพนังกานด้วย
- Cursor AI ออกแบบระบบการเพิ่มพนักงานใหม่ > มองเห็นระบบนี้ได้เฉพาะ Role : HR , Admin เท่านั้น
- Cursor AI ออกแบบระบบของการนำเข้าข้อมูลของพนักงานในจำนวนที่เยอะๆ ได้สามารถรองรับการนำเข้าพนักงานเป็นรูปแบบไฟล์ Excel 
- Cursor AI จะต้องทำ Dashboard สำหรับการสรุปพนักงานได้ว่าตอนนี้พนักงาน ทำงานอยู่กี่คน ลาออกกี่คน วิเคราะห์ข้อมูลของตำแหน่งงานจำนวนพนักงาน
  1. ทำระบบกราฟแท่ง วิเคราะห์ว่าในระยะเวลารอบ 6 เดือนที่ผ่านมาถึงปัจจุบันมีพนักงานเข้าทำงานแต่ละเดือนกี่คน พนักงานลาออกกี่คน Cursor AI ออกแบบเป็นกราฟแท่ง + กราฟเส้นเพื่อให้ดูข้อมูลง่าย
  2. Cursor AI ออกแบบระบบการดูข้อมูลของพนักงานว่าในอีกช่วง 90 วันข้างหน้าจะมีพนักงานคนไหนจะต้องประเมินการผ่านทดลองงาน โดยให้ระบบอิงข้อมูลจากวันที่เริ่มงาน
  3. Cursor AI ออกแบบระบบการเเสดงข้อมูลแต่ละตำแหน่งภายในบริษัทจำนวนพนักงานกี่คนบ้าง

**Cursor AI จะต้องออกแบบ API ของการแสดงข้อมูลตรงนี้ให้ระบบทำงานได้อย่างรวดเร็วไม่หนักหน้าเว็บ โหลดข้อมูลมาเท่าที่จำเป็นเท่านั้น และจะต้องทำ Document สำหรับเก็บข้อมูลด้วยว่าหน้านี้ใช้ API เส้นไหนมาทำงาน ระบบทำงานยังไง ดึงข้อมูลยังไงมีผลดียังไงบ้าง
**Cursor AI ออกแบบ DataBase จะต้องทำ Document สำหรับหัวตารางว่าแต่ละชื่อแทนด้วยอะไรจากข้อมูลที่ผมมี
